## Installation
To install and use the contracts, follow these steps:
1. **Clone the repository:**
    ```bash
    git clone xxx
    cd xxx
    ```
2. **Install dependencies:**
    ```bash
    npm install
    ```
3. **Compile the contracts:**
    ```bash
    npx hardhat compile
    ```
4. **Run the tests:**
    ```bash
    npx hardhat test
    ```

5. **Run the deploy:**
    ```bash
    npm deploy --network <<NETWORK>>
    ```
