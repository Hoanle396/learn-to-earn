# DOP Marketplace API

## Description

A API repository integrate by [NestJs](http://nestjs.com)

## Installation

```bash
# First, copy env file:
$ cp env.example .env
```

```bash
# Install dependencies 
$ pnpm install
```

## Running the app

```bash
# development
$ pnpm run start

# watch mode
$ pnpm run dev

# production mode
$ pnpm run start:prod
```
